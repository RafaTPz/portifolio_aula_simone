Sistema de Estoque e Agendamento para Barbearia

Tipo: Projeto Integrador Acadêmico

Sistema web desenvolvido para auxiliar no controle de estoque de produtos e no agendamento de horários de clientes em uma barbearia. O objetivo foi centralizar a gestão do negócio em uma única plataforma.

Funcionalidades:

  • Cadastro e controle de produtos em estoque
  • Registro de entrada e saída de itens
  • Agendamento de horários com controle de disponibilidade
  • Painel administrativo para gerenciamento

Tecnologias: PHP, Laravel e SQLITE

Sistema de Controle de Aplicação de Medicamentos - ONG Animal

Tipo: Projeto Integrador Acadêmico

Aplicação desenvolvida para auxiliar uma ONG no controle de aplicação de medicamentos em animais resgatados, garantindo organização e acompanhamento correto dos tratamentos.

Funcionalidades:

  • Cadastro de animais
  • Registro de medicamentos aplicados
  • Controle de datas e frequência de aplicação
  • Histórico completo por animal

Tecnologias: Java, JavaFX e PostgresSQL

Aplicação de Reconhecimento de Voz

Tipo: Projeto Pessoal

Pequeno projeto experimental desenvolvido com React utilizando biblioteca de reconhecimento de voz para capturar comandos por áudio e convertê-los em texto em tempo real.

Funcionalidades:

  • Captura de áudio pelo microfone
  • Conversão de voz para texto
  • Exibição dinâmica do texto na interface

Tecnologias: React, JavaScript, Biblioteca de Speech Recognition

Objetivo: Explorar APIs modernas e ampliar conhecimento em front-end
